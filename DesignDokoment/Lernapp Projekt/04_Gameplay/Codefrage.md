 
## Beschreibung  
Der Nutzer bekommt eine kleine Programmieraufgabe und muss eine kurze Code-Lösung eingeben.  
  
Die Aufgabe besteht meist aus 1–2 Zeilen Code und dient dazu, das aktive Programmieren zu üben.  
  
---  
  
## Bildschirm  
Der Bildschirm zeigt eine Aufgabenbeschreibung und ein Eingabefeld für Code.  
  
Optional kann ein Teil des Codes bereits vorgegeben sein.  
  
---  
  
## Elemente  
- Aufgabenbeschreibung (Text)  
- Code-Eingabefeld  
- Button "Ausführen" oder "Prüfen"  
- Feedback (richtig/falsch Anzeige)  
- optional: Beispiel oder Hinweis  
  
---  
  
## Benötigte Funktionen  
- Nutzer kann Code eingeben  
- System überprüft den Code  
- Vergleich mit der richtigen Lösung  
- Anzeige von "richtig" oder "falsch"  
- Weiterleitung zur nächsten Aufgabe  
  
---  
  
## Beispiel  
  
Aufgabe:  
Erstelle eine Variable namens "x" mit dem Wert 5.  
  
Richtige Lösung:  
x = 5  
  
## Darstellung  
  
Aufgabe:  
Erstelle eine Variable namens "x" mit dem Wert 5.  
  
[ Code-Eingabe ]  
  
> ____________  
  
[ Prüfen ]  
  
→ Nutzer schreibt Code  
→ System überprüft Eingabe  
→ Feedback wird angezeigt
