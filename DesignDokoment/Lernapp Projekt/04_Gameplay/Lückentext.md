  
## Beschreibung  
Der Nutzer bekommt einen Satz mit einer oder mehreren Lücken.  
Diese Lücken müssen mit den richtigen Begriffen ausgefüllt werden.  
  
Das System dient dazu, Wissen aktiv anzuwenden und Begriffe zu festigen.  
  
---  
  
## Bildschirm  
Der Bildschirm zeigt einen Satz mit einer sichtbaren Lücke.  
Unter oder in der Lücke befindet sich ein Eingabefeld.  
  
---  
  
## Elemente  
- Satz mit Lücke(n)  
- Eingabefeld (Textfeld)  
- Button "Bestätigen"  
- Feedback (richtig/falsch Anzeige)  
  
---  
  
## Benötigte Funktionen  
- Nutzer kann Text eingeben  
- System überprüft die Eingabe  
- Anzeige von "richtig" oder "falsch"  
- Vergleich mit der richtigen Lösung  
- Weiterleitung zur nächsten Aufgabe  
  
---  
  
## Beispiel  
  
Aufgabe:  
Eine Variable speichert einen _______.  
  
Richtige Antwort:  
Wert  
  
## Darstellung  
  
Eine Variable speichert einen [ _______ ]  
  
[ Eingabe: __________ ]  
  
[ Bestätigen ]  
  
→ Nutzer gibt Antwort ein  
→ System überprüft Eingabe  
→ Feedback wird angezeigt
