
## Beschreibung  
Der Nutzer bekommt eine Frage mit vier Antwortmöglichkeiten.  
  
## Ablauf  
- Frage wird angezeigt  
- Nutzer wählt eine Antwort aus  
- System überprüft die Antwort  
- Ergebnis wird angezeigt (richtig oder falsch)  
  
## Elemente  
- Frage (Text)  
- 4 Antwortmöglichkeiten (Buttons)  
- Feedback (richtig/falsch Anzeige)  
  
## Funktionen  
- Auswahl einer Antwort  
- Überprüfung der Antwort  
- Rückmeldung an den Nutzer  
- Weiterleitung zur nächsten Aufgabe

## Beispiel  
  
Frage:  
Was ist eine Variable in der Programmierung?  
  
Antwortmöglichkeiten:  
A) Ein fester Wert  
B) Ein Speicherort für Daten  
C) Ein Fehler im Code  
D) Ein Programm  
  
Richtige Antwort:  
B) Ein Speicherort für Daten  
  
## Darstellung  
  
[ Frage: Was ist eine Variable? ]  
  
[ A ] Ein fester Wert  
[ B ] Ein Speicherort für Daten  
[ C ] Ein Fehler im Code  
[ D ] Ein Programm  
  
→ Nutzer klickt auf eine Antwort  
→ System zeigt "richtig" oder "falsch"
