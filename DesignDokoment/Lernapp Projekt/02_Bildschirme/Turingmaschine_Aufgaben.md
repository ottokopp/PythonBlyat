    
## Bildschirm  
Der Bildschirm zeigt ein Raster aus Vierecken, das eine vereinfachte Turing-Maschine darstellt.  
Ein Cursor zeigt die aktuelle Position auf dem Raster.  
Der Nutzer kann den Cursor über das Spielfeld bewegen.  
  
## Elemente  
- Raster (Spielfeld aus Vierecken)  
- Cursor (markiertes Feld)  
- Steuerbuttons (oben, unten, links, rechts)  
- Aufgabenbeschreibung (Textfeld)  
  
## Benötigte Funktionen  
- Cursor kann sich in vier Richtungen bewegen (oben, unten, links, rechts)  
- Spielfeld reagiert auf die Bewegung des Cursors  
- Aufgabe wird überprüft (z. B. Ziel erreicht)  
- Rückmeldung bei Erfolg oder Fehler  
- Fortschritt wird gespeichert
