 
## Bildschirm  
Der Bildschirm zeigt das Hauptmenü der App.  
Hier kann der Nutzer verschiedene Funktionen auswählen.  
  
## Elemente  
- Profilname  
- Anzeige des User-Levels  
- Fortschrittsanzeige  
- Button "Start"  
- Button "Quickstart"  
- Button "tägliche Aufgaben"  
- Button "Einstellungen"  
  
## Benötigte Funktionen  
- Anzeige des aktuellen Nutzers  
- Klick auf "Start" öffnet die Levelübersicht  
- Klick auf "Quickstart" startet eine zufällige Aufgabe  
- Klick auf "tägliche Aufgaben" startet die Tagesaufgabe  
- Klick auf "Einstellungen" öffnet das Einstellungsmenü
