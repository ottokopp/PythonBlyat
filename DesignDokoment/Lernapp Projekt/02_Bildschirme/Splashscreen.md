
## Bildschirm  
Beim Start der App wird ein kurzer Startbildschirm angezeigt.  
Dieser zeigt das Logo der App auf einem einfachen Hintergrund.  
  
## Elemente  
- App-Logo  
- Hintergrund  
- optional Ladeanimation  
  
## Benötigte Funktionen  
- Splashscreen wird beim Start der App angezeigt  
- Anzeige für wenige Sekunden (ca. 2–3 Sekunden)  
- danach automatische Weiterleitung zum Startbildschirm
